### 1️⃣ Install dependencies
From inside your project folder:
```
cd ~/prasanth-poultry
npm install
```
This will:
> Install Vite && Create node_modules/.bin/vite

2️⃣ Run build again
```
npm run build
```
✅ Now it should work and create the dist/ folder.

## ✅ After Successful Build
You should see:
```
dist/
├── index.html
├── assets/
```
